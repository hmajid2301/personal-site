## Color

You can adjust the colour of the icon if you pass the `color` prop.

<Preview>
  <Story name="Color">
    <div>
      <ThemeIcons hover="blue-500" color="gray-500" size="3xl" />
      <ThemeIcons hover="blue-500" color="purple-500" size="3xl" />
    </div>
  </Story>
</Preview>

## Hover

You can adjust the colour on hover of the icon if you pass the `hover` prop.

<Preview>
  <Story name="Hover">
    <div>
      <ThemeIcons hover="green-500" color="gray-800" size="3xl" />
      <ThemeIcons hover="purple-500" color="gray-800" size="3xl" />
    </div>
  </Story>
</Preview>
