## Background

You can adjust the background colour of the component by passing the `background` prop.

<Preview>
  <Story name="Background">
    <div>
      <Hero background="gray-200" text="Hello my name is Haseeb." />
      <Hero background="yellow-500" text="Hello my name is Haseeb." />
    </div>
  </Story>
</Preview>

## Color

You can adjust the colour of the text by passing the `color` prop.

<Preview>
  <Story name="Color">
    <div>
      <Hero
        background="gray-200"
        color="black"
        text="Hello my name is Haseeb."
      />
      <Hero
        background="indigo-100"
        color="blue-500"
        text="Hello my name is Haseeb."
      />
    </div>
  </Story>
</Preview>

## Text

You can adjust the main text of the component by passing the `text` prop.

<Preview>
  <Story name="Text">
    <div>
      <Hero
        background="gray-200"
        color="black"
        text="Hello this is a new message."
      />
      <Hero
        background="indigo-100"
        color="blue-500"
        text="Another random message."
      />
    </div>
  </Story>
</Preview>

## showParticles

If you want to show the particles you can pass the `showParticles` prop.

<Preview>
  <Story name="Show Particles">
    <div>
      <Hero
        background="gray-200"
        color="black"
        text="Hello my name is Haseeb."
        showParticles
      />
      <Hero
        background="indigo-100"
        color="blue-500"
        text="Hello my name is Haseeb."
        showParticles
      />
    </div>
  </Story>
</Preview>

## particleColor

You can adjust the particle colour if you pass the `particleColor` prop.

<Preview>
  <Story name="Particle Color">
    <div>
      <Hero
        background="gray-200"
        color="black"
        text="Hello my name is Haseeb."
        showParticles
        particleColor="#367ee9"
      />
      <Hero
        background="indigo-100"
        color="blue-500"
        text="Hello my name is Haseeb."
        showParticles
        particleColor="#ff00ff"
      />
    </div>
  </Story>
</Preview>
