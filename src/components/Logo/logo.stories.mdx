## Colour

You can change the default colour of the logo by passing the `color` prop. Where the colour is
the name of a colour from tailwindcss. You can see the default list
[here](https://tailwindcss.com/docs/customizing-colors/#default-color-palette).

<Preview>
  <Story name="color">
    <div>
      <Logo color="gray-500" />
      <Logo color="blue-500" />
    </div>
  </Story>
</Preview>

## Accent

You can adjust the accent (tags) color by passing the `accent` prop. Where again the
colour is from tailwindcss. You can see the default list
[here](https://tailwindcss.com/docs/customizing-colors/#default-color-palette).

<Preview>
  <Story name="accent">
    <div>
      <Logo accent="gray-500" color="blue-500" />
      <Logo accent="gray-500" color="black" />
    </div>
  </Story>
</Preview>

## Size

You can also adjust the size of the logo by passing the `size` prop. Where once again
this size if set using tailwindcss examples. You can see the default list
[here](https://tailwindcss.com/docs/font-size/).
