# Icon

  <Story

## Background

You can adjust the background colour of icon the by passing the `background` prop.

<Preview>
  <Story name="Background">
    <div>
      <Icon
        background="indigo-500"
        color="blue-500"
        hoverColor="blue-500"
        icon={<Search />}
      />
      <Icon
        background="gray-800"
        color="purple-500"
        hoverColor="blue-500"
        icon={<Search />}
      />
    </div>
  </Story>
</Preview>

## Color

You can adjust the colour if you pass the `color` prop.

<Preview>
  <Story name="Color">
    <div>
      <Icon color="blue-500" hoverColor="blue-500" icon={<Search />} />
      <Icon color="purple-500" hoverColor="blue-500" icon={<Search />} />
    </div>
  </Story>
</Preview>

## Hover Color

You can adjust the colour if you pass the `hoverColor` prop.

<Preview>
  <Story name="Hover Color">
    <div>
      <Icon color="blue-500" hoverColor="black" icon={<Search />} />
      <Icon color="purple-500" hoverColor="green-500" icon={<Search />} />
    </div>
  </Story>
</Preview>
