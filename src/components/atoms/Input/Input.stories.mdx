## Background

You can adjust the background colour of input the by passing the `background` prop.

<Preview>
  <Story name="Background">
    <div>
      <Input background="gray-800" color="white" />
      <Input background="indigo-500" color="white" />
    </div>
  </Story>
</Preview>

## Color

You can adjust the colour of the input if you pass the `color` prop.

<Preview>
  <Story name="Color">
    <div>
      <Input background="gray-500" color="blue-500" />
      <Input background="gray-500" color="purple-500" />
    </div>
  </Story>
</Preview>
