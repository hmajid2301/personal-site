## OnChange

You can call a function everytime the input is changed by passing the `onChange` function.

<Preview>
  <Story name="OnChange">
    <Input
      onChange={(e) => console.log(e.target.value)}
      placeholder="Search Bar"
    />
  </Story>
</Preview>

## Placeholder

You can adjust the placeholder text if you pass the `placeholder` prop.
